> Вода, активность, сон — всё в одном месте!

## 🌟 Основные возможности

### 💧 Водный трекер
- ✅ Установка дневной цели (1.5л, 2л, 2.5л, 3л или свободный ввод: 500–8000 мл)
- ✅ **Автоматический еженедельный сброс** статистики с учётом часового пояса
- ✅ **Полное логирование** добавлений воды (защита от потери данных при сбоях)
- ✅ Гибкие напоминания:
  - ⏰ **По интервалу** (каждые 1–5 часов)
  - 🤖 **Умный режим** (напомнит, если не пил 3+ часа)
- ✅ Быстрые кнопки (250, 330, 450, 750, 1000 мл) + ручной ввод
- ✅ Прогресс-бар с мотивационными сообщениями
- ✅ Защита от спама (минимум 15 минут между записями)
- ✅ Безопасные лимиты: 5л (разумный), 8л (жёсткий)

### 💪 Трекер физической активности
- ✅ Расширенная библиотека упражнений с фильтрацией:
  - 6 категорий (Силовые, Кардио, Растяжка, Ходьба, Зарядка, Баланс)
  - 3 уровня сложности (Новичок, Средний, Продвинутый)
- ✅ **Видео-демонстрация упражнений** (опционально)
- ✅ Избранные упражнения и пагинированные списки
- ✅ Статистика выполнения (всего, за неделю, ТОП-5 популярных)
- ✅ Умные лимиты времени между выполнениями:
  - Одно упражнение: не чаще раза в 15 минут
  - Разные упражнения: не чаще раза в 5 минут
- ✅ Админ-панель для управления упражнениями
- ✅ Гибкие напоминания (по интервалу / умный режим)

### 😴 Трекер сна
- ✅ Настройка времени отбоя и подъёма (быстрые кнопки + свой ввод)
- ✅ Кнопка «Ложусь спать» / «Проснулся» для логирования сна
- ✅ История сна с продолжительностью и качеством
- ✅ Минимальное время сна: 30 минут
- ✅ Напоминания «пора спать» и «время просыпать»
- ✅ Тихие часы (напоминания не беспокоят в время сна)

### 🎮 Геймификация (XP и уровни)
- ✅ Начисление XP за выполнение упражнений, добавление воды, сон
- ✅ 10 уровней с титулами (Новичок → Мастер ЗОЖ)
- ✅ Глобальная таблица лидеров (TOP-20)
- ✅ Личный XP-профиль с прогресс-баром до следующего уровня
- ✅ Уведомления о повышении уровня

### 🤖 ИИ-анализ профиля
- ✅ Локальная LLM (модель `gemma3:4b` через [Ollama](https://ollama.com))
- ✅ Анализ показателей воды, активности и сна
- ✅ Персонализированные советы с учётом статистики пользователя
- ✅ Формат вывода: 5 разделов (Вода, Активность, Сон, Главный совет, Мотивация)

### 📊 Статистика пользователя
- ✅ Вода (цель, сегодня, неделя), активность (цель, сегодня, неделя), сон (среднее, количество за неделю)
- ✅ XP, уровень, позиция в топе
- ✅ Инлайн-клавиатура: Таблица лидеров | Мой XP | ИИ-анализ

### 👨‍⚕️ Система обращений (тикеты)
- ✅ Категории: Техподдержка и Персональные консультации
- ✅ Диалог с администратором в чате
- ✅ Пагинация, статусы, прикрепление фото

### ⚙️ Настройки пользователя
- ✅ Часовой пояс (Россия: МСК-1 … МСК+9)
- ✅ Цели, напоминания и интервалы для каждого трекера

### 🛠️ Админ-панель
- ✅ **Статистика**: всего, активных, новых, подсчёт по каждому трекеру
- ✅ **Поиск пользователя** по ID или @username
- ✅ **Просмотр профиля**: статус, дата регистрации, активность, XP, позиция в топе
- ✅ **Бан/разбан** пользователя с уведомлением (бан блокирует все действия)
- ✅ **Редактирование XP**: прибавить / вычесть любое количество XP пользователю
- ✅ **Массовая рассылка** с предпросмотром
- ✅ **Управление администраторами** (добавить/удалить по ID/username)
- ✅ **Автоматическая цензура**: словарь + AI-проверка (Detoxify, опционально)
- ✅ **Просмотр и ответы** на обращения пользователей
- ✅ **Управление упражнениями**: добавление, редактирование, статистика

## 🚀 Установка и запуск

### 1️⃣ Клонирование

```bash
git clone https://github.com/твой-юзербейм/zozhnik-bot.git
cd zozhnik-bot
```

### 2️⃣ Виртуальное окружение

```bash
python -m venv .venv

# Windows:
.venv\Scripts\activate

# Linux/macOS:
source .venv/bin/activate
```

### 3️⃣ Зависимости

```bash
pip install -r requirements.txt
```

### 4️⃣ Владельцы бота (обязательно)

Создай файл `owners.json` в корне проекта:

```json
[
  6295868318,
  123456789
]
```

> 📝 **Где взять ID?** Напиши [@userinfobot](https://t.me/userinfobot) в Telegram

### 5️⃣ Переменные окружения

Создай файл `.env`:

```env
token_bot=YOUR_TELEGRAM_TOKEN_HERE
MEDIA_STORAGE_CHANNEL_ID=-1001234567890
```

> 🔑 **Получить токен:** Напиши [@BotFather](https://t.me/BotFather) → /newbot

> 📺 **Настройка медиа-канала:**
> 1. Создай приватный канал в Telegram
> 2. Добавь бота в канал как администратора
> 3. Перешли любое сообщение из канала боту [@userinfobot](https://t.me/userinfobot)
> 4. Скопируй ID канала (формат: `-100XXXXXXXXXX`)
> 5. Укажи ID в `.env` как `MEDIA_STORAGE_CHANNEL_ID`

### 6️⃣ (Опционально) ИИ-анализ через Ollama

```bash
# Установить Ollama: https://ollama.com/download
ollama pull gemma3:4b
ollama serve
```

> Если Ollama не запущен, кнопка «ИИ-анализ» будет отсутствовать, остальное работает без изменений.

### 7️⃣ Запуск

```bash
python main.py
```

✅ **Готово!** Бот работает и готов к использованию
